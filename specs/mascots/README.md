# Mascot designs — original, vetted, ready to generate

15 original creature designs, each with a complete image prompt. Generate any of them
into base art with:

```bash
scripts/gen_mascots.sh 80 specs/mascots/ember.json   # ~0.6% of a weekly Codex quota each
```

Every design was written to be **original and CC0-safe** — no resemblance to any existing
character — and to survive being drawn as a 192×208 sprite (clean silhouette, no detached
parts, nothing finer than ~4px). They are the design half of the pipeline; the art itself has
not been generated yet.
